CRITICAL THREAT DETECTED

FAUX#ELEVATE

O FAUX#ELEVATE é uma ameaça avançada de exfiltração de dados e mineração. O ataque inicia-se através de currículos falsos (Fake CVs) ofuscados em VBScript, desenhados para contornar filtros de rede corporativos e comprometer credenciais empresariais.

Vetor de Infeção

VBScript ofuscado em documentos de recrutamento

Componente: mservice.exe

Implementação do minerador XMRig para exploração de CPU

Componente: mozilla.vbs

Script dedicado ao roubo de credenciais do navegador Firefox

Persistência no Kernel

Uso do driver WinRing0x64.sys para maximizar recursos

Ação Evasiva

Desativação forçada do Microsoft Defender e Controlo de Conta de Utilizador (UAC)

Manual de Defesa Ativa

1. Neutralização de Binários

Localize e encerre o processo mservice.exe. Remova o script mozilla.vbs das pastas de inicialização (Startup) e chaves de registro 'Run'.

2. Expurgo de Driver de Kernel

O driver WinRing0x64.sys permite acesso Ring 0. Remova o serviço via CMD administrativo:

sc stop "WinRing0" && sc delete "WinRing0"

3. Restauração de Segurança

Reative o Microsoft Defender e limpe as exclusões de caminhos criadas pelo malware no registro do sistema.
