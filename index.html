CRITICAL THREAT: FAUX#ELEVATE

GUIA DE DEFESA ATIVA

[01] NEUTRALIZAÇÃO DO VETOR INICIAL

O ataque inicia-se via documentos .VBS ofuscados. A primeira medida é desabilitar o motor de execução de scripts nativo do Windows em pastas temporárias.

cmd // disable wscript.exe / cscript.exe via GPO

[02] IDENTIFICAÇÃO DE BINÁRIOS MALICIOSOS

Monitore e encerre os processos mservice.exe (XMRig Miner) e verifique a presença do script mozilla.vbs na pasta Startup. Estes são responsáveis pela exfiltração de credenciais e consumo de CPU.

[03] REMOÇÃO DE PERSISTÊNCIA NO KERNEL

O FAUX#ELEVATE utiliza o driver WinRing0x64.sys para privilégios de Ring 0. Verifique o diretório de drivers e utilize o comando abaixo para remover serviços não assinados:

sc stop "WinRing0" && sc delete "WinRing0"
